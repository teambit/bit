#### Use `ComponentCard` to display an overview of a component in a card.

`ComponentCard` accepts an array of `plugins`. Each `plugin` accepts a `component` prop with the component details.

Plugins are presented in specific places in the card:

- `topLeft` - presented in the top left corner
- `topRight` - presented in the top right corner
- `preview` - presented at the top section of the card
- `bottomLeft` - presented in the bottom left corner
- `bottomRight` - presented in the bottom right corner

```tsx live
() => {
  const previewPlugin = {
    preview: ({ component }) => (
      <img
        style={{ width: '100%', height: '100%' }}
        src="https://storage.googleapis.com/static.bit.dev/Community/app-components/homepage.jpg"
      />
    ),
  };
  const plugins = [
    {
      topLeft: [
        ({ component }) => {
          return <img src="https://static.bit.dev/extensions-icons/react.svg" />;
        },
      ],
      topRight: [
        ({ component }) => {
          return (
            <div
              style={{
                background: '#ffc640',
                borderRadius: 8,
                fontSize: 10,
                padding: '2px 4px',
                textTransform: 'uppercase',
                fontWeight: 'bold',
              }}
            >
              {component.id.name.split('-').pop()}
            </div>
          );
        },
      ],
      bottomLeft: [
        ({ component }) => {
          return <img style={{ width: 16 }} src="https://static.bit.dev/bit-logo.svg" />;
        },
      ],
      bottomRight: [
        ({ component }) => {
          return <img src="https://static.bit.dev/extensions-icons/react.svg" />;
        },
      ],
    },
  ];
  return <ComponentCard plugins={[...plugins, previewPlugin]} component={componentMock} />;
};
```

:::tip Please notice
If multiple plugins have a preview, the card will display the preview of the last plugin that was passed.
:::
