import { useState } from 'react';
import { SearchInput } from './search-input';

A Input component used for search use cases.  
Use `onSubmit` prop to pass a function when the search icon is clicked.  
The `onSubmit` function will also be triggered when Enter key is pressed.  
Supports all props from native html input.

### Component usage

```js live
() => {
  const [value, setvalue] = useState('');
  return (
    <SearchInput value={value} onChange={(e) => setvalue(e.target.value)} onSubmit={() => alert('on icon click')} />
  );
};
```
