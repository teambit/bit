import { LevelIcon } from './level-icon';

## Icon Component with various alert levels

- info
- warning
- success
- error

### Component usage

```js
import { LevelIcon } from '@teambit/design.ui.elements.level-icon';

...
<LevelIcon level="info" />
...
```

### Using props to change the icon

Modify the alert level of icon to see the icon change:

```js live
<LevelIcon level="info" />
```
