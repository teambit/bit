### Overview

An Ellipsis div, which dislays cut off text with an ellipsis (...) if it doesnt fit into the element's width


long string with ellipsis displayed:
```js live
  <ThemeCompositions style={{ minHeight: 90 }}>
    <Ellipsis style={{ width: 100 }}>Looooooooooooooong string</Ellipsis>
  </ThemeCompositions>
```
