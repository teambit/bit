## Static Content Page in React

Docs for React aspect

### Component usage

Use your component in any React file:

```js
import { React } from '@teambit/aspect-docs.React';

<React />;
```

### Render with theme and React providers

```js
import { ThemeContext } from '@teambit/documenter.theme.theme-context';
import { ReactLayout } from '@teambit/ui.React-layout';
import { React } from '@teambit/aspect-docs.React';

<ThemeContext>
  <ReactLayout>
    <React />
  </ReactLayout>
</ThemeContext>;
```
