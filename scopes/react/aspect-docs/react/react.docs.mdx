## Static Content Page in MDX

Docs for React aspect

### Component usage

Use your component in any MDX file:

```js
import { React } from '@teambit/aspect-docs.React';

<React />;
```

### Render with theme and React providers

```js
import { ThemeContext } from '@teambit/documenter.theme.theme-context';
import { MDXLayout } from '@teambit/ui.mdx-layout';
import { React } from '@teambit/aspect-docs.React';

<ThemeContext>
  <MDXLayout>
    <React />
  </MDXLayout>
</ThemeContext>;
```
