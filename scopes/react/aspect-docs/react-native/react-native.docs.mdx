## Static Content Page in MDX

Docs for ReactNative aspect

### Component usage

Use your component in any MDX file:

```js
import { ReactNative } from '@teambit/aspect-docs.react-native';

<ReactNative />;
```

### Render with theme and ReactNative providers

```js
import { ThemeContext } from '@teambit/documenter.theme.theme-context';
import { MDXNativeLayout } from '@teambit/ui.mdx-layout';
import { ReactNative } from '@teambit/aspect-docs.react-native';

<ThemeContext>
  <MDXNativeLayout>
    <ReactNative />
  </MDXNativeLayout>
</ThemeContext>;
```
