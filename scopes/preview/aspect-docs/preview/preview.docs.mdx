## Static Content Page in Preview

Docs for Preview aspect

### Component usage

Use your component in any Preview file:

```js
import { Preview } from '@teambit/aspect-docs.preview';

<Preview />;
```

### Render with theme and Preview providers

```js
import { ThemeContext } from '@teambit/documenter.theme.theme-context';
import { PreviewLayout } from '@teambit/ui.Preview-layout';
import { Preview } from '@teambit/aspect-docs.preview';

<ThemeContext>
  <PreviewLayout>
    <Preview />
  </PreviewLayout>
</ThemeContext>;
```
